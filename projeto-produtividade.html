Projeto: Gerenciador de Produtividade

Este projeto foi desenvolvido para solucionar a necessidade de um acompanhamento de produtividade mais dinâmico, transparente e motivador dentro do setor. Utilizando o Google Apps Script, criei uma aplicação web completa que funciona como um sistema de "gamificação", onde os servidores podem lançar suas atividades diárias e acompanhar seu desempenho.

A aplicação é integrada a uma Planilha Google, que atua como banco de dados para armazenar os lançamentos, os cadastros de servidores e as pontuações de cada atividade. O frontend foi construído com HTML e CSS, e todo o backend (servidor web, comunicação com a planilha, lógica de negócios) é gerenciado pelo JavaScript do lado do servidor do Google Apps Script.

Tecnologias Utilizadas

  • Google Apps Script
  • JavaScript (ES6+)
  • HTML5
  • CSS3
  • Planilhas Google (API)
  • Google Charts API

Principais Funcionalidades

Acesso ao Sistema: Tela de autenticação individual para cada servidor, criada com mecanismo de segurança algorítmo SHA256

Lançamento de Atividades: Formulário simples e intuitivo para o registro diário.

Dashboard Individual: Gráfico de evolução da produtividade diária para auto-avaliação.

Dashboard Geral (Gamificação): Ranking "Top 10" que possibilita um acompanhamento da produtividade dos servidores.
